# lyrae-php

Lyrae project for PHP

## Pre-Installation
## 
<ul>
  <li>php manager composer</li>
  <li>php-cli</lib>
</ul>

## Install command
### php-cli
```shell==
sudo apt install php7.4-cli
php -v
```
### composer
```shell===
 curl -sS https://getcomposer.org/installer | php
 php composer.phar
 mv composer.phar composer
 chmod +x composer 
 mv composer /usr/local/bin/
 sudo mv composer /usr/local/bin/
 composer -v
```
